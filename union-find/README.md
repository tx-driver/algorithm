# Union-Find

## 概要
* '#', '.'で構成されたテーブルをUnion-Findで探索
* 上下左右に1個以上連結した'#'を併合
* rootを-1, '.'は-2として管理
* 一度findすると経路を圧縮（スタックを利用）

## 使用例
* 4x4テーブルを探索
* 併合後のテーブルを出力(4x4個の要素を持つ1次元リスト)

```
>python union_find.py
4 4
##..
.#.#
..##
##..
[1, 5, -2, -2, -2, -1, -2, 11, -2, -2, 11, -1, 13, -1, -2, -2] # 圧縮前
[5, 5, -2, -2, -2, -1, -2, 11, -2, -2, 11, -1, 13, -1, -2, -2] # 圧縮後
```
